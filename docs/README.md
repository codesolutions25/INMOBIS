# Componentes Estandarizados para Módulos CRUD

Este proyecto incluye un conjunto de componentes genéricos para estandarizar la implementación de módulos CRUD en la aplicación. Estos componentes permiten mantener una consistencia visual y de comportamiento en toda la aplicación, facilitando el desarrollo y mantenimiento.

## Componentes Disponibles

### 1. EntityDetailView
Componente para mostrar detalles de entidades con un diseño estandarizado.

### 2. EntityForm
Componente para crear formularios con un diseño estandarizado.

### 3. Formateadores para columnas
Conjunto de funciones para formatear datos en las columnas de las tablas.

## Documentación

Para más detalles sobre cómo implementar nuevos módulos CRUD utilizando estos componentes, consulta la [Guía de Implementación](./GUIA_IMPLEMENTACION.md).

## Ejemplo Práctico

El módulo de Propiedades (`src/modules/propiedades`) implementa todos estos componentes y puede servir como referencia para nuevos desarrollos.

## Mejores Prácticas

1. Mantén la consistencia visual en todos los módulos
2. Utiliza los componentes genéricos para facilitar el desarrollo
3. Sigue los patrones establecidos en el módulo de Propiedades
4. Documenta cualquier extensión o modificación de los componentes genéricos
